吉他查询 仅支持欧服

  1. 基础指令
    ojita [物品名]*[数量(可选参数)]

    /ojita 凡晶石*100

  2. 进阶指令
    ojita [物品名]*[数量]
    --use [esi|tycoon|esi_cache]
    --history [follow|only|none]

    --use [esi|tycoon|esi_cache |ceve]
    选择一个查询接口
    默认esi_cache每30min刷新一次
    你可以选择esi或者tycoon查询最新价格
    --history [follow|only|none]
    选择历史价格图片状态
    默认follow，附在价格后面
    你可以选择只发送历史价格折线图或者不发送

  3. 翻译物品
    trans [物品名称] --num [数量] (num可选)

    trans 毒蜥级
    --num查询翻译列表返回多少项，默认6项

zkilboard功能系列

  1. 查询角色zkb信息
    zkb [角色名]

    /zkb Redeovg

  2. 查询指定军团签名排行
    rank [军团名] (支持缩写)

    /rank THXFC

  3. 订阅高价值击杀
    /高价值击杀订阅 add|remove -l/--limit[value]

    /高价值击杀订阅 add -l20000000000
    请一定把[]去掉，是-l不是-1

  4. 订阅指定角色km
    /击杀订阅
    add|remove --corp|--char [name] -a[value] -v[value]

    /击杀订阅 add --char Redeovg
    进阶用法：
    /击杀订阅 add --char Redeovg -a15000000 -v30000000
    -a是订阅的击杀最低限度推送价格
    默认15m，最低10m
    -v是订阅的损失最低限度推送价格
    默认30m，最低10m
    这个命令请写在一行，这里写不下了所以换行了
    看清楚是订阅的角色（char）还是军团（corp）
    订阅联盟需要联系管理员
    恶意订阅直接拉进黑名单
